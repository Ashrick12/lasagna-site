# Lasagna

**Get Opinions on Lasagna**

## Installation

1. **Clone this repository**

```
git clone https://github.com/Ashrick12/lasagna-site.git
```

2. **Open the folder in terminal**

```
cd lasagna-site
```

3. **Run the server**

```
py server.py
```

4. **Open the website on a web browser** `127.0.0.1:8000`
